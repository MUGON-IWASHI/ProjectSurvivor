# Project Survivor

## 概要

Project Survivor は Unreal Engine 5.8 と C++ を用いて制作した3Dアクションゲームです。

プレイヤーは迫ってくる敵を倒しながら経験値を獲得し、レベルアップを繰り返して生き残ることを目的としています。

ゲームプログラマーへの転職を目指し、C++によるゲームロジックの実装やBlueprintとの連携など、実践的なゲーム開発を目的として制作しました。

---

## 制作背景

ゲーム業界への転職を目指し、

・C++によるゲーム開発
・Unreal Engineの理解
・Blueprintとの役割分担
・Gitによるソース管理

を実践的に学ぶことを目的として制作しました。

---

## ゲーム内容

プレイヤーを操作し、一定時間生き残ることを目的とした3Dアクションゲームです。

敵を倒して経験値を獲得するとレベルアップし、プレイヤーが徐々に強化されます。

敵は一定時間ごとにスポーンし、ゲームオーバーになるまでゲームが続きます。

## 開発で工夫した点

### C++とBlueprintの役割分担

ゲームロジックはC++で実装し、調整しやすい部分はBlueprintで管理することで保守性を意識しました。

### 学習しながら実装

実装を急ぐのではなく、仕組みを理解したうえで実装することを意識しました。

問題が発生した際は原因を調査し、一つずつ解決しながら開発を進めました。

### Gitによる開発

機能ごとにコミットを分け、変更履歴を管理しながら開発を行いました。
